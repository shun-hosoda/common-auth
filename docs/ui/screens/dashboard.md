## 情報設計

### 表示要素と優先度

| 優先度 | 要素 | 表示条件 | 目的 |
|---|---|---|---|
| 高 | ウェルカムカード（名前・メール・ロールバッジ） | 全員 | ログイン状態の確認 |
| 中 | サイドナビ「ユーザー管理」メニュー | tenant_admin / super_admin | 管理操作への導線 |
| 低 | UserDropdown（セキュリティ設定・ログアウト） | 全員 | アカウント操作 |

**削除・非表示にする要素**:
- セッション情報（有効期限・スコープ）→ 一般ユーザーには不要
- デバッグパネル（プロフィールJSON）→ 画面には出さない
- MFA・ログアウトの画面内ボタン → UserDropdown に移動済み

## コンポーネントマッピング

| UI要素 | 使用コンポーネント | props |
|---|---|---|
| トップバー全体 | `TopBar`（インライン実装） | — |
| アバター + ドロップダウン | `UserDropdown` | `initial`, `name`, `email`, `roleLabel`, `roleColor`, `items` |
| サイドナビ（PC） | `SideNav` | `items`, `currentPath`, `onNavigate` |
| ハンバーガードロワー（SP） | `MobileDrawer` | `open`, `onClose`, `items` |
| ウェルカムカード | `WelcomeCard` | `initial`, `name`, `email`, `roleLabel`, `roleColor` |
| ロールバッジ | `Badge` | `label`, `color`（→ design-system.md のロールカラー参照） |
| ユーザーアイコン | `Avatar` | `initial`, `size` |

---

## 状態一覧

| 状態 | 表示内容 |
|---|---|
| 通常 | ウェルカムカード + ナビ |
| ユーザー名取得失敗 | `—`（フォールバック） |
| ロール未取得 | 「一般ユーザー」バッジを表示 |

---

## SP対応詳細

| 要素 | PC | SP（768px以下） |
|---|---|---|
| サイドナビ | 左カラム固定表示 | 非表示（ハンバーガーでドロワー） |
| TopBar | ロゴ + UserDropdown | ☰ + ロゴ + UserDropdown（縮小） |
| UserDropdown ユーザー名 | 表示 | 省略（アバターのみ） |
| コンテンツ余白 | `32px` | `16px` |
| ウェルカムカード | 横並び | 横並び（wrap）|
