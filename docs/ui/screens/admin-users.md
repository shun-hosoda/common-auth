# 画面仕様：ユーザー管理

| 項目 | 内容 |
|---|---|
| パス | `/admin/users` |
| アクセス条件 | `tenant_admin` または `super_admin` |
| 最終更新 | 2026-03-02 |

---

## 情報設計

### 表示要素と優先度

| 優先度 | 要素 | 表示条件 | 目的 |
|---|---|---|---|
| 高 | ユーザー一覧テーブル | 全管理者 | ユーザー把握 |
| 高 | ユーザー登録ボタン | 全管理者 | 新規ユーザー追加 |
| 中 | 検索フィールド | 全管理者 | 目的ユーザーの検索 |
| 中 | 編集ボタン（行内） | 全管理者 | 情報更新 |
| 低 | 有効/無効トグル | 全管理者 | アカウント制御 |

### データスコープ

| ロール | 閲覧・操作対象 |
|---|---|
| `tenant_admin` | 自テナント（`tenant_id`が一致）のユーザーのみ |
| `super_admin` | 全テナントのユーザー |

## コンポーネントマッピング

| UI要素 | 使用コンポーネント | 備考 |
|---|---|---|
| ユーザー一覧（PC） | `Table` | `columns`, `rows`, `onRowClick` |
| ユーザーカード（SP） | カスタムCard | Table の代替 |
| 検索フィールド | `FormField` + `input[type=search]` | — |
| 登録・編集モーダル | `Modal` | `isOpen`, `onClose`, `title` |
| ロールバッジ | `Badge` | ロールカラーはdesign-system.md参照 |
| 有効/無効表示 | `Badge` | 有効:`color-success` / 無効:`color-text-muted` |
| 登録ボタン | `Button` | `variant="primary"`, `icon="＋"` |
| 編集ボタン | `Button` | `variant="ghost"`, `size="sm"`, `icon="✏"` |
| ローディング | `Spinner` | API呼び出し中 |
| データなし | `EmptyState` | ユーザーが0件 |
| エラー | `Alert` | `type="error"` |

## フォーム仕様

### ユーザー登録フォーム

| フィールド | 必須 | バリデーション |
|---|---|---|
| メールアドレス | ✅ | メール形式 |
| 表示名 | ✅ | 1文字以上 |
| 仮パスワード | ✅ | 8文字以上 |
| 仮パスワード（次回変更必須） | — | チェックボックス（デフォルトON） |
| ロール | ✅ | `user` / `tenant_admin`（super_adminのみ） |
| テナント | ✅（super_adminのみ） | テナント選択 |

### ユーザー編集フォーム

| フィールド | 必須 | 備考 |
|---|---|---|
| 表示名 | ✅ | — |
| メールアドレス | ✅ | — |
| ロール | ✅ | `tenant_admin` は `user` のみ変更可 |
| 状態（有効/無効） | — | トグル |
| パスワードリセット | — | 別アクション（確認ダイアログ付き） |

---

## 状態一覧

| 状態 | 表示内容 |
|---|---|
| ローディング中 | Spinner（テーブル領域） |
| データあり | テーブル / カード一覧 |
| データなし | EmptyState「ユーザーがいません」 |
| 検索結果なし | EmptyState「該当するユーザーが見つかりません」 |
| API エラー | Alert `type="error"` + 再試行ボタン |
| 登録・編集成功 | Alert `type="success"`（自動消去） |
